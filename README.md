### Setting up TypeScript for Production/Project
`npx tsc --init` > creates `tsconfig.ts`
`npm init -y` > creates `package.json`
set `"outDir":"./" to "./dist"` to generate js in that dir.
`npx ts -w` (watchmode) > autoupdates .js in the `outDir`
